# [Nexus Mutual](https://app.nexusmutual.io/)

## Getting Started

**Requirements**: `Node >= 10`  
**Install**: `npm i`  
**Compile**: `npx truffle compile`  
**Test**: `npm test`  
**Deploy**: `npx truffle deploy`
